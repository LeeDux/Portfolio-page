NC-News Backend Project

Developed an API to manage application data using JavaScript, PostgreSQL, Node.js, and Jest. I implemented endpoints to retrieve articles, topics, users, and comments, along with features for sorting, filtering, adding comments, and updating votes. The project emphasized error handling, testing, and building a scalable, efficient backend service, using Test-Driven Development (TDD) throughout. You can view this project on my Github profile.

NC-News Frontend Project

Developed the front-end for NC News, a social news aggregation platform where users can view, comment, and vote on articles, categorised by topics. Utilised JavaScript, HTML, CSS, and React to implement features such as article sorting, comment deletion, and error handling. Integrated with a backend API for data interaction, creating a dynamic and responsive user interface with state management and routing. You can view this project on my Github profile.

Corporate-Conquest Phaser Game

Developed a dynamic 2D top-down rogue-like web browser game using Phaser.js, React, Redux, JavaScript, HTML, and CSS. Integrated a Node.js and Express backend with Passport.js for secure user authentication, enabling leaderboard functionality and personalised user experiences. The project focused on creating a visually engaging, responsive front end with scalable backend services, emphasising modular design, gameplay logic, and seamless data interaction. You can view this project on my Github profile.
