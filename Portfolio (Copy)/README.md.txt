#My Portfolio Website
=====================

##about this project
---------------------

In this project I will put to practice what I have learned so far on my journey to becoming a web-developer by building a portfolio website.

###The website will include:

*A photo of me
*About me section
*Skills Section
*Contact information
*Project examples
*Social links to my linked in and GitHub

###Skills used to make this website:

*HTML for the structure of the website
*CSS to style the page
*javaScript to make the website interactive
*Implementation of markdown for the README file
*Git bash/ GitHub for the version control of the website
*Visual Studio Code for the coding environment.
*Use of Google Chrome Dev Tools.
